Проект: Skygen

Документація:


Проєкт


Для перегляду сторінок проекту, скористайтесь, будь ласка, навігаційною панеллю зліва.


Збірка

Команди для збірки:

npm run dev

Запускає режим розробника

npm run build

Запускає режим збірки проекту в папку dist

npm run zip

Запускає режим збірки проекту в ахів

npm run ftp

Запускає режим збірки проекту і відправлення на ftp-сервер

npm run page -- --page=назва_файлу --name=назва_сторінки

Створення нової сторінки і створення всіх необхідних файлів для роботи сторінки в проекті
